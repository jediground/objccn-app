大家好！

这期的内容全是关于动画的。本期是第 12 期，下个月就是我们创刊一周年的纪念日了！上个月在 Android 的世界里愉快地转了一圈后，本期我们回归到 Objective-C。我们试图覆盖动画的方方面面——从 UIView 动画，Core Animation，一直到从零打造一个你自己定义的动画。我们还会讨论 View Controller 和 Collection View 有关的动画，以及如何让动画变得可以交互。

再一次地，感谢所有为本期内容做出贡献的嘉宾作者们。首先，Robert Böhnke 向我们展示了动画是如何运作的，以及在创建动画过程中应该注意些什么。Nick Lockwood 则举例展示了如何为 Layer 中的自定义属性添加动画，以及如何给不可见的对象添加动画。接下来，Joachim Bondo 撰文解释了应该如何给自定义的 View Controller 添加转场动画。David Rönnqvist 的文章详述了 UIView 动画的工作原理，以及应该如何来实现这种动画。Engin Kurutepe 跟我们分享了如何创建自定义的 Collection View 动画。最后，Florian 和 Chris 写了一篇关于交互式动画的文章，并展示了几种不同的实现方法。

从柏林发来诚挚的祝福！

Chris，Daniel，和 Florian。
