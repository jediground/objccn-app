欢迎来到 objc.io 的第 22 期。这个月，我们社区中一些在知名公司就职的开发者将分享他们的工作经验。

一开始 Brent Simmons 将介绍在Omni 集团中应用开发的流程。接下来 Artsy 团队写下了开源工作环境对公司文化的积极影响. Kyle Van Essen 描述了 Square 如何调整它的管理流程以适应团队的快速发展和质量要求的提高。Ashley Nelson-Hornstein 带领我们走了一遍 Dropbox 的代码审核流程。最后，Adam Ernst 向我们介绍了 Components ，这是由 Facebook 的新鲜事团队开发的响应式第三方库，它能让 UI 代码更容易编写，也更容易维护。

我们要感谢所有贡献者。和上个月一样，Ole Begemann 帮助作者们做了一些调整。谢谢 Ole！
