欢迎来到 objc.io 的第 24 期，本期的内容是音频。

在今年，我们已经先后聚焦了两个主题：相片与视频，所以在这个月补充了媒体系列之前缺失的部分：音频。

Jack Schaedler 最近写了一篇惊艳的 信号处理介绍，使本期的开头阐述了什么是音频处理程序中的“Hello World!”：追踪单声道音乐演奏的音高。(文中的精美插图也是他的作品！)

接下来，Chris 讲述了使用 Swift 进行函数式信号处理，而 Aaron 则从另一个角度审视了这个主题，面向产品的音效设计。最后，如果你还是个新手，对苹果平台上层出不穷的音频框架感到困惑的话，Daniel 与 Florian 提供了一份高层级 API 一览。

来自柏林的美好祝福，

Chirs，Daniel，与 Florian。
